Família de Fontes com CSS

Combinação de Fontes Seguras

São combinações de fontes de uma mesma categoria. Por exemplo: sans-serif.
Para verificar as combinações clique aqui.

Fontes Genéricas

            body {
                font-family: Arial, Helvetica, sans-serif;
            }
Dessa maneira o navegador procurará a fonte Arial. Caso não encontrar, tentará a Helvetica. E, por fim, se não achar, utilizará uma outra sem serifa.
