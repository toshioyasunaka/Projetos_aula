Diferença entre ID e CLASS

Nunca repita o nome de ID

Nesse caso é melhor utilizar-se da propriedade class.

Não dê nomes para suas classes e ID's pela forma

Exemplo: class="red", não dê esse nome, pois se um cliente pedir para mudar a cor de algum item com essa propriedade, o nome não fará mais sentido.

Dê nomes pela funcionalidade

Exemplo: class="basico". A ofazer um catálogo com vários cursos, os cursos básicos terão seu nome em vermelho. Nesse caso criamos uma propriedade class="basico" para todos os cursos básicos, e colorimos a fonte de vermelho. Ao invés de criar uma propriedade class="vermelho" para todos os cursos básicos que serão coloridos de vermelho.

Nomenclatura no CSS

No CSS geralmente nomeamos as tags com ID da seguinte forma: 


            h2#curso_HTML {
                
            }
Pois são itens únicos e saber que estrutura é, contribui para uma melhor organização.

Já as class sem a estrutura antes do nome designado para ela. Ficando da seguinte forma: 


            .basico {
                
            }
Isso porque podem ser várias tags diferentes a receber a mesma classificação.
