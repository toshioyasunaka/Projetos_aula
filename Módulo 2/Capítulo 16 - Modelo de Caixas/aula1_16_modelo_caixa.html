Modelo de Caixas

Anatomia de uma caixa

  • Height: altura do elemento;
  • Width: largura do elemento;
  • Border: linha que simboliza os limites do elemento;
  • Outline: contorno fora da borda, fora do elemento;
  • Padding: preenchimento para que a borda fique com uma certa distância do conteúdo;
  • Margin: espaço externo à borda;

Tipos de caixa

  • Box-level: sempre ocupa a largura total da tela. Se for inserido um novo elemento depois de um elemento box-level, ficará na linha seguinte.

    • Exemplos:

    • < div >
    • < h1 - h6 >
    • < p >
    • < main >
    • < header >
    • < nav >
    • < article >
    • < aside >
    • < footer >
    • < form >
    • < video >

  • Inline-level: não pula para a próxima linha. Nem antes e nem depois do conteúdo ser inserido. Continua o texto de onde estava seu fim.

    • Exemplos:

    • < span >
    • < a >
    • < code >
    • < small >
    • < strong >
    • < em >
    • < sup - sub >
    • < label >
    • < button >
    • < input >
    • < select >
